# RG Plomberie

Site vitrine premium développé pour RG Plomberie, entreprise spécialisée en plomberie, chauffage, climatisation, VMC et dépannage dans le Rhône.

## Aperçu du projet

Ce projet présente une refonte complète d’un site vitrine professionnel avec une identité visuelle immersive, moderne et responsive.

L’objectif est de valoriser les prestations de l’entreprise, rassurer les visiteurs et faciliter la prise de contact via un formulaire clair, une page dépannage, une galerie de réalisations et une carte interactive de zone d’intervention.

## Fonctionnalités principales

- Page d’accueil immersive
- Présentation de l’entreprise
- Pages prestations détaillées
- Galerie de réalisations
- Section avant / après
- Page dépannage
- Formulaire de contact et demande de devis
- Carte interactive de zone d’intervention
- Mode clair / mode sombre
- Menu responsive mobile
- Sitemap XML
- Robots.txt
- Mentions légales
- Design responsive desktop, tablette et mobile
